#Js Spa Template

This template was made to build SPA with js vanilla more easier.

##*Prerequisites*
* NodeJs

##*Commands*
* **npm run build**. Generate a minified bundle
* **npm run dev**. Generate a bundle
* **npm install** inside test folder to install packages for testing
* **npx webpack serve** to run webpack devserver

##*Template structure*
* app
    - dist
    - node_modules
    - src
        - components
            - home  
                * home.js
                * home.html
            - error-404
                * error-404.js
                * error-404.html
            * index.components.js
        - routes
            * index.routes.js
        - utils
        * app.js
        * index.html
    * .babelrc
    * .gitignore
    * package.json
    * readme.md
    * webpack.config.js
